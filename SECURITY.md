# Security Policy

## Reporting a vulnerability

Please report security issues privately by emailing:

- desmond.tan986@gmail.com


Include:

- Affected version/commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if any)

## Scope notes

This project can orchestrate agent-driven terminal actions. Treat any automation path that can run shell commands, inject pane input, or process untrusted message payloads as security-sensitive.

## Response target

- Initial acknowledgment: within 7 days
- Triage decision: within 14 days
